# Japanese Engine User Manual

Japanese Engine is a study dashboard for Module 0 / JE-001. It helps you build beginner Japanese through sentence patterns, verb forms, vocabulary recall, writing practice, and completion tracking.

## Quick Start

1. Open the app at `http://127.0.0.1:8000/`.
2. Choose a unit from the dark Japanese Engine dashboard.
3. Read the unit map and objective patterns.
4. Practice with the five loops generated for the selected engine.
5. Use the contextual form matrix, contextual pattern generator, and selected-engine checklist.
6. Write your own sentences in the writing box.
7. Click `Log practice` when you finish a unit practice session.

Japanese text includes hiragana readings in parentheses, for example:

## Main Screens

## 1. Engine Dashboard

The dark top section is the curriculum dashboard.

- `Japanese Engine`: the app title.
- `Module 0 / JE-001`: the current learning module.
- `Engine readiness`: your overall progress percentage.
- Unit cards: the ten study engines.

Click any unit card to switch the current study focus.

The units are:

1. Sentence Engine
2. Verb Engine
3. Particle Engine
4. Question Engine
5. Description Engine
6. Time Engine
7. Action Engine
8. Connection Engine
9. Expression Engine
10. Conversation Engine

## 2. Current Unit

The current unit appears below the dashboard.

The unit panel shows:

- The unit name.
- The unit goal.
- A short explanation of what to practice.
- Objective patterns or skills.

Use this section first. It tells you what the unit is training.

## 3. Practice Engine

The Practice Engine contains five learning loops for the currently selected engine. Use the numbered tabs `1` to `5` to move between them. When you choose a different engine, the loop titles, examples, and practice rows change with that engine.

The loops are:

- Vocabulary Recall
- Verb Transformation
- Sentence Transformation
- Pattern Substitution
- Conversation

Each row gives you Japanese, hiragana reading, and meaning or practice direction.

Recommended use:

1. Read the Japanese.
2. Read the hiragana if needed.
3. Say the phrase aloud.
4. Check the English meaning.
5. Try to produce the Japanese again without looking.

## 4. Verb Engine

The form matrix changes with the selected engine. Context Form Matrix shows verb forms, Description Engine shows adjective-style forms, and other engines show the forms most relevant to that engine.

It includes:

- Dictionary form
- Polite form
- Negative form
- Past form
- Te-form
- Potential form

Click the arrow button to move to the next verb.

Recommended use:

1. Read the main verb.
2. Say each form aloud.
3. Cover the Japanese and try to recall each form.
4. Use one form in an original sentence.

## 5. Context Pattern Generator

The contextual pattern generator helps you build original sentences from the selected engine.

Choose:

- Pattern mode
- Subject
- The selected engine's component
- The selected engine's verb or phrase

The app combines them into a sentence, such as:

`&#31169;&#12399;&#23398;&#26657;&#12391;&#21193;&#24375;&#12375;&#12414;&#12377;&#12290;&#65288;&#12431;&#12383;&#12375;&#12399;&#12364;&#12387;&#12371;&#12358;&#12391;&#12409;&#12435;&#12365;&#12423;&#12358;&#12375;&#12414;&#12377;&#12290;&#65289;`

Click the shuffle button to generate a new combination.

Use the writing box below it to write 3 to 5 original sentences from the pattern.

## 6. Portfolio Checklist

The Portfolio Checklist tracks larger completion goals.

Click `Mark complete` when you can confidently do the task. Click it again if you need to unmark it.

Checklist progress contributes to the Engine readiness score.

## Progress Tracking

The app saves progress in your browser on this computer.

Progress includes:

- Completed unit practice logs.
- Completed portfolio checklist items.

Click `Log practice` to mark the current unit as practiced.

Click the reset button if you want to clear all progress and start again.

## Suggested Daily Study Routine

Use this simple 20 to 30 minute routine:

1. Choose one unit.
2. Read the Bootloader Map objective patterns.
3. Do Vocabulary Recall for 5 minutes.
4. Do Verb Transformation for 5 minutes.
5. Generate 3 original sentences.
6. Write those sentences in the writing box.
7. Say each sentence aloud.
8. Click `Log practice`.

For deeper study, continue through all five practice loops before logging the unit.

## How To Study With Hiragana Readings

The app shows kanji and hiragana together.

Try this order:

1. Look at the kanji first.
2. Guess the reading.
3. Check the hiragana in parentheses.
4. Say the full phrase aloud.
5. Look away and repeat it from memory.

The goal is not only to read hiragana. The goal is to slowly recognize kanji words through repeated exposure.

## Troubleshooting

If the app does not open:

- Make sure the local server is running.
- Open `http://127.0.0.1:8000/`.
- If the page says connection refused, restart the app server.

If old text appears:

- Refresh the page.
- If needed, open `http://127.0.0.1:8000/?fresh=1`.

If progress looks wrong:

- Progress is saved in the browser.
- Using a different browser or clearing browser data can reset progress.
- Use the reset button only when you want to clear all saved progress.

## Best Practice

Do not rush through every unit. Japanese Engine works best when you repeat a small number of patterns until they feel automatic.

For each pattern, aim to:

- Read it.
- Say it.
- Change one word.
- Write a new sentence.
- Use it in a short answer.
