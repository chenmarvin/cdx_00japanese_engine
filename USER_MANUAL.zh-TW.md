# Japanese Engine 使用手冊（繁體中文）

Japanese Engine 是 Module 0 / JE-001 的日文學習儀表板。它協助你透過句型、動詞變化、單字回想、造句練習與完成度追蹤，建立初級日文基礎。

## 快速開始

1. 開啟 app：`http://127.0.0.1:8000/`。
2. 從深色的 Japanese Engine 儀表板選擇一個單元。
3. 閱讀目前單元的地圖與目標句型。
4. 使用依所選引擎變化的依所選引擎變化的五個學習循環進行練習。
5. 使用情境形式矩陣與情境句型產生器。
6. 在寫作區寫下自己的句子。
7. 完成一次單元練習後，點選 `Log practice`。

日文單字與句子會附上平假名讀音，例如：

## 主要畫面

## 1. Engine Dashboard

深色的上方區塊是課程儀表板。

- `Japanese Engine`：app 名稱。
- `Module 0 / JE-001`：目前學習模組。
- `Engine readiness`：整體完成度。
- 單元卡片：十個學習引擎。

點選任一單元卡片即可切換目前的學習主題。

十個單元如下：

1. Sentence Engine
2. Verb Engine
3. Particle Engine
4. Question Engine
5. Description Engine
6. Time Engine
7. Action Engine
8. Connection Engine
9. Expression Engine
10. Conversation Engine

## 2. Current Unit

目前單元會顯示在儀表板下方。

單元面板包含：

- 單元名稱。
- 單元目標。
- 練習重點說明。
- 目標句型或技能。

建議先閱讀這個區塊，了解本單元正在訓練什麼能力。

## 3. Practice Engine

Practice Engine 包含依所選引擎變化的依所選引擎變化的五個學習循環。使用數字分頁 `1` 到 `5` 切換。

五個循環如下：

- Vocabulary Recall
- Verb Transformation
- Sentence Transformation
- Pattern Substitution
- Conversation

每一列會提供日文、平假名讀音，以及英文意思或練習方向。

建議練習方式：

1. 先讀日文。
2. 需要時查看括號中的平假名。
3. 大聲唸出來。
4. 檢查英文意思。
5. 遮住日文後，再試著自己說出來。

## 4. Verb Engine

Verb Engine 一次顯示一個動詞。

它包含：

- 辭書形
- 丁寧形
- 否定形
- 過去形
- て形
- 可能形

點選箭頭按鈕即可切換到下一個動詞。

建議練習方式：

1. 讀出主要動詞。
2. 大聲唸每一個變化形。
3. 遮住日文，試著回想每個形式。
4. 選一個形式造一個自己的句子。

## 5. Context Pattern Generator

Context Pattern Generator 會協助你建立自己的原創句子。

你可以選擇：

- Subject
- Place / object
- Verb

app 會把它們組合成句子，例如：

`&#31169;&#12399;&#23398;&#26657;&#12391;&#21193;&#24375;&#12375;&#12414;&#12377;&#12290;&#65288;&#12431;&#12383;&#12375;&#12399;&#12364;&#12387;&#12371;&#12358;&#12391;&#12409;&#12435;&#12365;&#12423;&#12358;&#12375;&#12414;&#12377;&#12290;&#65289;`

點選 shuffle 按鈕可以產生新的組合。

下方的寫作區可以用來寫 3 到 5 個從這個句型延伸出來的原創句子。

## 6. Portfolio Checklist

Portfolio Checklist 用來追蹤較大的完成目標。

當你能有把握完成某項能力時，點選 `Mark complete`。如果之後想取消標記，可以再點一次。

清單完成度會影響 Engine readiness 分數。

## 進度追蹤

app 會把進度儲存在目前瀏覽器中。

進度包含：

- 已記錄練習的單元。
- 已完成的 portfolio checklist 項目。

點選 `Log practice` 可以標記目前單元已完成一次練習。

如果想清除所有進度並重新開始，請點選 reset 按鈕。

## 建議每日練習流程

可以使用這個 20 到 30 分鐘的流程：

1. 選擇一個單元。
2. 閱讀 Bootloader Map 的目標句型。
3. 做 Vocabulary Recall 5 分鐘。
4. 做 Verb Transformation 5 分鐘。
5. 產生 3 個原創句子。
6. 把句子寫在寫作區。
7. 大聲唸出每個句子。
8. 點選 `Log practice`。

如果想更深入練習，可以完成依所選引擎變化的依所選引擎變化的五個學習循環後再記錄單元。

## 如何使用平假名讀音

app 會同時顯示漢字與平假名。

建議順序：

1. 先看漢字。
2. 先猜讀音。
3. 再看括號中的平假名。
4. 大聲唸出完整片語。
5. 移開視線，再從記憶中重複一次。

目標不只是會讀平假名，而是透過反覆接觸，慢慢認得漢字單字。

## 疑難排解

如果 app 無法開啟：

- 確認本機伺服器正在執行。
- 開啟 `http://127.0.0.1:8000/`。
- 如果頁面顯示 connection refused，請重新啟動 app 伺服器。

如果看到舊內容：

- 重新整理頁面。
- 必要時開啟 `http://127.0.0.1:8000/?fresh=1`。

如果進度看起來不正確：

- 進度儲存在瀏覽器中。
- 換瀏覽器或清除瀏覽器資料可能會重設進度。
- 只有在想清除所有進度時才使用 reset 按鈕。

## 最佳練習方式

不要急著跑完所有單元。Japanese Engine 最適合用來反覆練習少量句型，直到句型變得自然。

每個句型都建議做到：

- 讀出來。
- 說出來。
- 替換一個單字。
- 寫一個新句子。
- 用它回答一個短問題。
