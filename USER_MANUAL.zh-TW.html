Japanese Engine 使用手冊

Japanese Engine 是 Module 0 / JE-001 的日文學習儀表板，目標是幫助台灣學習者先建立日文句子引擎，再進入 N5、N4、N3 與更高階學習。

快速開始

  1. 開啟 app：http://127.0.0.1:8000/。
  2. 從 Japanese Engine 儀表板選擇一個單元。
  3. 閱讀目前單元的目標與句型。
  4. 使用五個學習循環練習。
  5. 使用動詞矩陣與句型產生器。
  6. 在寫作區寫下自己的句子。
  7. 完成練習後，點選 Log practice。

主要畫面

  • Engine Dashboard：顯示 10 個 Japanese Engine 單元與 Engine readiness。
  • Current Unit：顯示目前單元的學習目標、說明與核心句型。
  • Practice Engine：提供 Vocabulary Recall、Verb Transformation、Sentence Transformation、Pattern Substitution、Conversation 五種練習。
  • Verb Engine：顯示動詞的辭書形、丁寧形、否定形、過去形、て形、可能形。
  • Pattern Generator：選擇 subject、place/object、verb 後，自動組成句子。
  • Portfolio Checklist：追蹤較大的能力完成項目。

10 個學習單元

  1. Sentence Engine
  2. Verb Engine
  3. Particle Engine
  4. Question Engine
  5. Description Engine
  6. Time Engine
  7. Action Engine
  8. Connection Engine
  9. Expression Engine
  10. Conversation Engine

建議每日練習流程

  1. 選擇一個單元。
  2. 閱讀 Bootloader Map 的目標句型。
  3. 做 Vocabulary Recall 5 分鐘。
  4. 做 Verb Transformation 5 分鐘。
  5. 產生 3 個原創句子。
  6. 把句子寫在寫作區。
  7. 大聲唸出每個句子。
  8. 點選 Log practice。

如何使用平假名讀音

app 會在日文單字與句子旁提供平假名讀音。建議先看漢字或假名，自己猜讀音，再查看括號中的讀音，最後大聲唸出完整句子。

進度追蹤

目前進度儲存在使用中的瀏覽器。完成單元練習與 Portfolio Checklist 項目後，Engine readiness 會更新。

進度備份與跨裝置同步

如果你在 Android phone 和 tablet 之間交替使用，可以用 app 上方的 Export 與 Import 功能手動同步進度。

  1. 在目前使用的裝置按 Export，下載 japanese-engine-progress-YYYY-MM-DD.json。
  2. 把這個 JSON 檔傳到另一台裝置，例如用 Nearby Share、Google Drive、Syncthing、USB 或其他檔案傳輸方式。
  3. 在另一台裝置開啟 app，按 Import，選擇剛剛匯出的 JSON 檔。
  4. 匯入後，completed units 與 checklist items 會合併到目前裝置的進度。
  5. floating writing pad 的文字不會被直接覆蓋；如果匯入內容不同，app 會追加到現有內容下方。

更新 app code 通常不會清除進度，但換瀏覽器、換網址或 port、清除瀏覽器資料，或改用另一台裝置時，local progress 可能不同。建議定期 Export 進度作為備份。

疑難排解

  • 如果無法開啟 app，確認 static server 是否正在執行。
  • 如果看到 ERR_CONNECTION_REFUSED，代表 127.0.0.1:8000 沒有伺服器在跑。
  • 如果畫面沒有更新，請重新整理或使用強制重新整理。
  • 如果進度看起來不正確，可以使用 reset 按鈕重新開始。

最佳練習方式

不要急著跑完所有單元。Japanese Engine 的重點是把少量高頻句型練到自然，能讀、能說、能替換單字、能寫出自己的句子，最後能在短對話中使用。
