Japanese Engine 使用手冊

Japanese Engine 是 Module 0 / JE-001 的日文學習儀表板，目標是幫助台灣學習者先建立日文句子引擎，再進入 N5、N4、N3 與更高階學習。

快速開始

  1. 開啟 app：http://127.0.0.1:8000/。
  2. 從 Japanese Engine 儀表板選擇一個單元。
  3. 閱讀目前單元的目標與句型。
  4. 使用五個學習循環練習。
  5. 使用動詞矩陣與句型產生器。
  6. 在寫作區寫下自己的句子。
  7. 完成練習後，點選 Log practice。

主要畫面

  • Engine Dashboard：顯示 10 個 Japanese Engine 單元與 Engine readiness。
  • Current Unit：顯示目前單元的學習目標、說明與核心句型。
  • Practice Engine：提供 Vocabulary Recall、Verb Transformation、Sentence Transformation、Pattern Substitution、Conversation 五種練習。
  • Verb Engine：顯示動詞的辭書形、丁寧形、否定形、過去形、て形、可能形。
  • Pattern Generator：依 Pattern mode 選擇 subject、location/object/destination、verb，產生不會任意錯配的句子。
  • Unit Visual Examples：Bootloader Map 的三個範例會依照目前選擇的 engine category 更新。
  • Unit Learning Items：選擇 engine category/unit 時，該 unit 的 objectives 會自動加入 floating writing pad，作為可直接練習的 prompts。
  • Component Copying：點選 unit、practice loop、Verb Engine 的 Next，或 Pattern Generator 的任何選項時，目前語言元件會自動加入 floating writing pad。
  • Portfolio Checklist：追蹤較大的能力完成項目。
  • Floating Writing Pad：可移動、可調整大小，也可以用 Close 隱藏，再用 Writing pad 重新開啟。
  • Finish Markers：完成 objective、practice row、conjugation form 或 generated pattern 的練習後，可以點選圓形標記。

10 個學習單元

  1. Sentence Engine
  2. Verb Engine
  3. Particle Engine
  4. Question Engine
  5. Description Engine
  6. Time Engine
  7. Action Engine
  8. Connection Engine
  9. Expression Engine
  10. Conversation Engine

各 Engine 的學習重點

  • Sentence Engine：先把日文看成可重複使用的句子框架。練習 topic、身分判斷、否定、所在位置與基本動作句，目標是不逐字翻譯也能自己組出簡單句。
  • Verb Engine：把動詞練成可以移動、可以變化的系統。比較一段、五段與不規則動詞，並反覆唸出辭書形、丁寧形、否定形、過去形、て形、可能形。
  • Particle Engine：不要只背助詞定義，而是看助詞在句子裡的工作。練習 は 表話題、が 表焦點、を 表目的語、に 表時間/目標、で 表動作場所或手段。
  • Question Engine：把陳述句轉成問句與回答。練習 yes/no 問句、誰・何・どこ・いつ 等疑問詞，以及短而自然的回答。
  • Description Engine：學會描述人、地方、物品與所有關係。先用 い形容詞、な形容詞、の 連接做出名詞片語，再放進完整句子。
  • Time Engine：把動作放進時間軸。練習具體時間、頻率、過去動作、未來意圖與期間，讓句子可以描述行程與日常習慣。
  • Action Engine：把高頻動詞變成生活輸出。練習移動、吃、喝、讀、工作、學習、買、看、睡覺、起床等動作，並放入可使用的句子。
  • Connection Engine：把短句連成想法。練習順序、轉折、原因、て形連接與名詞並列，讓輸出不再只是零散句子。
  • Expression Engine：練習意圖與人際互動。包含想要、允許、請求、能力、邀請與禮貌表達，讓初級日文可以真的拿來跟人互動。
  • Conversation Engine：把前面所有能力整合進對話。練習輪流說話、聽不清楚時的修復語、shadowing、role play、短日記與慢速初級對話。

建議每日練習流程

  1. 選擇一個單元。
  2. 閱讀 Bootloader Map 的目標句型。
  3. 使用自動加入 floating writing pad 的 learning items 作為練習清單。
  4. 點選 practice loop tab、Verb Engine 的 Next，或 Pattern Generator 的選項，把目前語言元件送到 floating writing pad。
  5. 做 Vocabulary Recall 5 分鐘。
  6. 做 Verb Transformation 5 分鐘。
  7. 使用 Pattern Generator 產生安全的句型 scaffold。
  8. 把 3 到 5 個原創句子寫在 floating writing pad。
  9. 大聲唸出每個句子。
  10. 完成個別項目後，點選 finish marker。
  11. 點選 Log practice。

如何使用平假名讀音

app 會在日文單字與句子旁提供平假名讀音。建議先看漢字或假名，自己猜讀音，再查看括號中的讀音，最後大聲唸出完整句子。

進度追蹤

目前進度儲存在使用中的瀏覽器。完成單元練習與 Portfolio Checklist 項目後，Engine readiness 會更新。

進度備份與跨裝置同步

如果你在 Android phone 和 tablet 之間交替使用，可以用 app 上方的 Export 與 Import 功能手動同步進度。

  1. 在目前使用的裝置按 Export，下載 japanese-engine-progress-YYYY-MM-DD.json。
  2. 把這個 JSON 檔傳到另一台裝置，例如用 Nearby Share、Google Drive、Syncthing、USB 或其他檔案傳輸方式。
  3. 在另一台裝置開啟 app，按 Import，選擇剛剛匯出的 JSON 檔。
  4. 匯入後，completed units 與 checklist items 會合併到目前裝置的進度。
  5. floating writing pad 的文字不會被直接覆蓋；如果匯入內容不同，app 會追加到現有內容下方。

更新 app code 通常不會清除進度，但換瀏覽器、換網址或 port、清除瀏覽器資料，或改用另一台裝置時，local progress 可能不同。建議定期 Export 進度作為備份。

疑難排解

  • 如果無法開啟 app，確認 static server 是否正在執行。
  • 如果看到 ERR_CONNECTION_REFUSED，代表 127.0.0.1:8000 沒有伺服器在跑。
  • 如果畫面沒有更新，請重新整理或使用強制重新整理。
  • 如果進度看起來不正確，可以使用 reset 按鈕重新開始。

最佳練習方式

不要急著跑完所有單元。Japanese Engine 的重點是把少量高頻句型練到自然，能讀、能說、能替換單字、能寫出自己的句子，最後能在短對話中使用。
