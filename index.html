Quick Start

  1. Choose a unit from the dark Japanese Engine dashboard.
  2. Read the current unit map and objective patterns.
  3. Practice with the five learning loops.
  4. Use the verb matrix and pattern generator.
  5. Write 3 to 5 original sentences in the writing box.
  6. Click Log practice when you finish a practice session.

Reading Support

Japanese words and sentences show hiragana readings in parentheses, such as 学校（がっこう）.

Main Tools

  • Practice Engine: five loops for recall, transformation, substitution, and conversation.
  • Verb Engine: one verb through dictionary, polite, negative, past, te-form, and potential forms.
  • Pattern Generator: combine subject, place/object, and verb into original sentences.
  • Portfolio Checklist: mark larger learning goals complete.

Progress

Progress is saved in this browser. The readiness score combines logged unit practice and completed checklist items. Use the reset button only when you want to start over.
